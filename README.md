A Stylelint config for emochan.

## Install

Install package:
```
npm install --save-dev stylelint stylelint-config-emochan
```

Then, add `stylelint-config-emochan` into configuration file.

```js
module.exports = {
  extends: ['stylelint-config-emochan'],
}
```
